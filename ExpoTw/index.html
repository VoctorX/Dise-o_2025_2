Definición

Tailwind CSS es un framework de CSS de tipo "utility-first" que proporciona clases de bajo nivel para construir diseños personalizados directamente en el HTML. A diferencia de otros frameworks CSS que ofrecen componentes predefinidos, Tailwind se centra en ofrecer herramientas fundamentales que permiten construir diseños únicos sin escribir CSS personalizado.

La versión 4 de Tailwind CSS introduce mejoras significativas en rendimiento, una API más simplificada y mejor integración con herramientas de desarrollo modernas, manteniendo la filosofía original de desarrollo rápido y eficiente.

Tailwind CSS te permite construir diseños personalizados sin abandonar tu HTML

Historia

Tailwind CSS fue creado por Adam Wathan y lanzado inicialmente en 2017. Su desarrollo surgió de la necesidad de encontrar un equilibrio entre CSS personalizado y frameworks rígidos. A lo largo de los años, ha evolucionado significativamente:

2017
Lanzamiento inicial de Tailwind CSS
2019
Versión 1.0 con mayor estabilidad y adopción comunitaria
2021
Versión 3.0 con soporte para JIT (Just-In-Time) compilation
2023
Versión 4.0 con arquitectura mejorada, mejor rendimiento y API simplificada

La versión 4 representa un salto cualitativo, optimizando el tamaño del bundle, mejorando la experiencia de desarrollo y simplificando la configuración.

Recursos y elementos a usar

Instalación

// CDN (Content Delivery Network)
<script src="https://cdn.tailwindcss.com"></script>
// npm (Node Package Manager)
1. Inicializa npm
npm init -y
2. Instala Tailwind CSS v4
npm install tailwindcss @tailwindcss/cli
3. Crea una carpeta css y dentro de ella un archivo styles.css que contenga:
@import "tailwindcss";
4. Regresa a la carpeta raíz del proyecto y ejecuta:
npx @tailwindcss/cli -i css/styles.css -o css/output.css --watch

Estructura y sintaxis

La sintaxis de Tailwind CSS sigue un patrón consistente y predecible:

[breakpoint:][state:]utility-[value]
Breakpoints
sm:, md:, lg:, xl:, 2xl:
States
hover:, focus:, active:, disabled:
Utilities
bg-, text-, p-, m-, w-, h-

Ejemplos de sintaxis:

bg-blue-500 // fondo azul 500
md:text-lg // texto grande en pantallas medianas y superiores
hover:bg-blue-700 // fondo azul 700 al pasar el cursor
p-4 md:p-8 // padding 4 en móvil, 8 en pantallas medianas
